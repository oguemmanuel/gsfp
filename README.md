## GSFP - School Feeding Programme Management Platform

A Next.js web app for managing a school feeding programme, with role-based dashboards for admins, teachers, students, and suppliers.

### Features

Role-based access for four user types: Admin, Teacher, Student, and Supplier. Authentication with sign in and registration. Dedicated dashboards per role for tracking and managing programme activity.

### Tech Stack

Frontend: Next.js and React. Backend: Node.js.

### Getting Started

Frontend: run `cd frontend`, then `npm install`, then `npm run dev`.

Backend: run `cd backend`, then `npm install`, then `npm start`.
